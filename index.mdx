Welcome to the GeoQuest documentation. GeoQuest is a geography guessing game built with **Next.js 14**, **React 18**, and **TailwindCSS** that uses the **novita.ai** LLaMA 3.1 model as an AI game host. A mystery country is randomly selected from 250+ countries, and players ask yes/no questions to figure out which one it is before running out of questions.

## Get up and running

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/andresparceromelo/actividad_app_react.git
    cd actividad_app_react
    ```
  </Step>
  <Step title="Install dependencies">
    ```bash
    npm install
    ```
  </Step>
  <Step title="Add your novita.ai API key">
    Copy the example environment file and add your key from the novita.ai dashboard.
    ```bash
    cp .env.example.api .env.local
    ```
    Edit `.env.local`:
    ```
    NOVITA_API_KEY=your_actual_novita_api_key_here
    ```
  </Step>
  <Step title="Start the development server">
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) to play.
  </Step>
</Steps>
