<Tip>
  You will need a free novita.ai API key to run GeoQuest. Visit [novita.ai](https://novita.ai/), sign in, and go to **Dashboard → API Keys → Create Key** to generate one at no cost.
</Tip>

## Set up the project

<Steps>
  <Step title="Check prerequisites">
    Make sure you have the following installed before continuing:

    - **Node.js 18 or later** — [nodejs.org](https://nodejs.org)
    - **npm** — bundled with Node.js
    - **A novita.ai API key** — required for the AI game host

    Verify your Node.js version:

    ```bash
    node --version
    ```
  </Step>

  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/andresparceromelo/actividad_app_react.git && cd actividad_app_react
    ```
  </Step>

  <Step title="Install dependencies">
    ```bash
    npm install
    ```

    This installs Next.js 14, React 18, TailwindCSS, and all associated tooling declared in `package.json`.
  </Step>

  <Step title="Configure your environment">
    Copy the example environment file:

    ```bash
    cp .env.example.api .env.local
    ```

    Open `.env.local` and replace the placeholder with your actual novita.ai key:

    ```bash
    NOVITA_API_KEY=your_actual_novita_api_key_here
    ```

    <Warning>
      Never commit `.env.local` to version control. It contains your secret API key. The repository's `.gitignore` excludes it by default — do not override that.
    </Warning>
  </Step>

  <Step title="Start the development server">
    ```bash
    npm run dev
    ```

    Open [http://localhost:3000](http://localhost:3000) in your browser. The home page will load, and navigating to `/game` starts a new round.
  </Step>
</Steps>

## How to play

<Steps>
  <Step title="A mystery country is selected automatically">
    When the game page loads, the app fetches all 250+ countries from the REST Countries API and randomly picks one. The country is stored secretly in state — you never see it until the game ends.
  </Step>

  <Step title="Ask yes/no questions">
    Type a question into the question input and submit it. Yes/no questions get the clearest answers:

    - "Is it in Europe?"
    - "Does it have a coastline?"
    - "Is it larger than France?"
    - "Does it speak Spanish?"

    The AI game host answers using the country's real data — capital, region, currency, and coordinates — without ever naming the country.
  </Step>

  <Step title="Track your remaining questions">
    A progress bar at the top of the game board shows how many of your 20 questions are used. When 5 or fewer questions remain, the bar turns red. Use your questions wisely.
  </Step>

  <Step title="Submit a guess when you're ready">
    Switch to the guess input and type the country name. Guessing costs one question slot whether you're right or wrong.
  </Step>

  <Step title="Win, lose, or play again">
    - **Correct guess** — You win. The full country card is revealed with its flag, capital, currency, and region.
    - **Wrong guess** — One question is deducted. Keep asking and try again.
    - **All 20 questions used** — Game over. The country is revealed so you can learn from it.

    Press **Play Again** to start a new round with a freshly selected country.
  </Step>
</Steps>
